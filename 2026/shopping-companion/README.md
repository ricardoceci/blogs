# Shopping Companion

A two-stage AI shopping agent with cross-session memory, inspired by [Shopping Companion (arXiv:2603.14864)](https://arxiv.org/abs/2603.14864).

Runs as a **chat web app** — open your browser and talk to it.

![Architecture](images/animation.svg)

▶ [Watch the demo](https://youtu.be/jRe34HBUry4)

## Stack

| Component | Service | Cost |
|---|---|---|
| LLM | Bedrock Claude Sonnet 4 | Pay per token |
| Embeddings | Bedrock Titan Embed v2 (via Mem0) | Pay per token |
| Cross-session memory | Mem0 free tier | Free |
| Product catalog | Shopify Storefront MCP | Free |
| Chat server | FastAPI + uvicorn | Free |
| Agent framework | Strands Agents | Free |

## Quickstart

```bash
git clone https://github.com/your-username/shopping-companion
cd shopping-companion
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

cp .env.example .env
# Set: SHOPIFY_STORE_DOMAIN, MEM0_API_KEY, AWS credentials

python scripts/index_history.py   # seed sample conversation history
python app.py                     # start server

# Open http://localhost:8000
```

## How it works

**First message in a session** — Stage 1 runs: retrieves style preferences from Mem0 memory and surfaces them for confirmation.

**User confirms or corrects** — the agent decides whether the response contains actual new preferences to save (not just "looks good"). Stage 2 runs: searches the live catalog with confirmed preferences.

**All subsequent messages in the same session** — preferences are already confirmed, Stage 1 is skipped entirely. The user can follow up ("now find a blazer", "what about shoes?") without re-confirming anything.

**Next session** — Stage 1 runs again, Mem0 returns the updated preference profile.

## Product backends

**Shopify (default)** — live store catalog via Storefront MCP. No auth needed.
```
PRODUCT_BACKEND=shopify
SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
```

**Local BM25** — sample catalog for testing without a live store.
```
PRODUCT_BACKEND=local
python scripts/seed_catalog.py
```
